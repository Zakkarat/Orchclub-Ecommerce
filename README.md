# Orchclub E-commerce

##### Даний проект передбачає створення інтернет-магазину для продажу квітів. 

### MVP:
  - Grid-based design.
  - Sign in/Sign up + Account managment.
  - Cart management.
  - Payments.
  
### Technologies:

#### Frontend:
  - Next.js
  - Redux
  
#### Backend:
  - Node.js
  - Koa.js
  - PostgreSQL(?)
  
### Current issues:
  - DB managment is not decided yet. (Postgres or NoSQL MongoDB)
  - Actual data parsing.
